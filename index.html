Обо мне

Backend developer с опытом работы 2 года. Сейчас активно работаю с языком Golang. В своей работе я стараюсь использовать собственные подходы к разработке, стараюсь сделать так, чтобы они соответствовали best practices языка и современным постулатам разработки.

Я специализируюсь на разработке API, для проектов использую Clean Architecture. Я стараюсь документировать свои проекты с помощью Swagger и покрывать тестами критические участки кода.
Кроме разработки API, я занимаюсь проектами, которые требуют знаний операционных систем. К примеру, один из моих проектов - это Execute Pipeline, он был написан с использованием подхода Test-driven development.

Также я знаком с подходом к организации приложений с помощью микросервисной архитектуры. Я разрабатывал приложение из двух сервисов: на Golang и Python с помощью grpc. Более подробное описание работы можно посмотреть ниже в блоке "Опыт".1.

Одна из самых запоминающихся моих работ - это потоковая асинхронная обработка видео. Передо мной стояла задача - сделать асинхронную нарезку видео на 10-ти секундные части с разным битрейтом. Также асинхронно загрузить эти части на s3 хранилище, таким образом, чтобы можно было во время последующего воспроизведения видео заменять поток данных на поток с другим разрешением (с помощью HTTP Live Streaming).

Сейчас я работаю на языке Golang, но также я работал с Django Framework на python, помимо этого, мне приходилось работать с языком Си. На языке Си я разрабатывал клиент-серверное приложение, работающее на web socket, приложение асинхронное, для эффективности обработки запросов я использовал механизм приоритетной очереди, этот проект тестировал с помощью CUnit.
